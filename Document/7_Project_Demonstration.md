# 7. Project Demonstration

## Demo Script

### Introduction
Welcome to HematoVision demonstration. This project classifies blood cells using transfer learning.

### Setup
- Run `python app.py` in Backend Code folder.
- Open browser to localhost:5000.

### Demo Steps
1. **Home Page:** Upload a blood cell image (e.g., eosinophil.jpg).
2. **Classification:** App processes and displays result: "Eosinophil" with confidence 98%.
3. **Report:** Detailed report generated.
4. **Educational Mode:** Feedback on classification.

### Scenarios Demonstration
- **Clinical:** Show integration potential.
- **Remote:** Simulate upload from remote location.
- **Educational:** Provide learning feedback.

### Q&A
- Accuracy achieved: 96%.
- Technologies: Flask, TensorFlow, MobileNetV2.

### Conclusion
HematoVision is ready for deployment and further enhancements.
